# Pokedex


# Overview
Pokedex is a mini mobile app built with expo and react native that allows users to browse,
search and favourite Pokemon using live data from the public [pokeAPI](https://pokeapi.co/)

| Screens | Description |
| :--- | :--- |
| Home | List of all pokemon with search function |
| Detail | Detail page for single pokemon |
| Favourites | List of users favourited pokemon |

# Set up

> [!WARNING]
> Requires [expo](https://expo.dev/)

1. Clone repo

```bash
git clone https://github.com/NoahAdamMacDonald/pokedex.git
cd assignment-two
```

2. Install Dependencies

```bash
npm install
```

3. Start

```bash
npx expo start
```
